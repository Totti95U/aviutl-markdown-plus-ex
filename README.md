# AviUtl MarkdownEX

[aviutl_browser](https://github.com/oov/aviutl_browser) を使って AviUtl で
Markdown を表示するスクリプト。

aviutl_browser 同梱の `Markdown.anm` を元に機能拡張を行ったものです。
